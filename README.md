# MySQL E-Commerce SQL Analysis Project

## Overview

This project contains SQL queries performed on a real-world e-commerce dataset using MySQL. The goal was to practice and strengthen SQL skills by working with relational data across multiple tables.

The project demonstrates practical use of SQL for data retrieval, filtering, aggregation, joins, subqueries, and analytical functions.

---

## Dataset Tables

The dataset contains the following tables:

- customers
- orders
- sellers
- products
- geolocation
- payments
- order_items

These tables simulate a complete e-commerce environment.
